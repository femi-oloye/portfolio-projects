Data Cleaning Using Messy Fifa21 Data

Introduction

This dataset is obtained from Kagle and the data was originally scraped from a website which made it messy, the data contains the name, photo URL, player URL, nationality, position, age, etc.

The dataset has the following problems:

  • wrong data types
  • newline characters in some columns
  • remove some characters and change their data types
  • Some players are high-valued and still underpaid
  • Missing values
  • outliers
  • wrong record format

from datetime import date

def calculate_year_of_birth(age):
    current_year = date.today().year
    year_of_birth = current_year-age
    return year_of_birth

Handling Outliers

There are different methods used to handle outliers, this methods are:
  • transforming outliers (Winsorization)
  • Using IQR Method
  • imputing outliers
  • Isolation forest (machine learning models)

For this study i will be transforming outliers using winsorization method. This limit extreme values in the data to reduce the effect of the outliers

In [175]:
from scipy.stats.mstats import winsorize
In [176]:
data['Value'] = data['Value'].astype(int, errors='ignore')
data['Value'] = winsorize(data['Value'], limits=[0.05, 0.05])
